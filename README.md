# Data Cleaning Projects using Python & pandas

## Overview

This repository contains multiple real-world data cleaning and preprocessing projects performed using Python and pandas.

The purpose of this repository is to practice and demonstrate practical data cleaning workflows used in:
- Data Analysis
- Machine Learning
- Business Intelligence
- Exploratory Data Analysis (EDA)

The datasets used in these projects contain realistic data quality issues such as:
- Missing values
- Duplicate rows
- Invalid datatypes
- Inconsistent categories
- Incorrect date formats
- Mixed text and numeric values
- Dirty business data

---

# Projects Included

## 1. Cafe Sales Dataset Cleaning

### Problems Solved
- Missing values
- Duplicate records
- Dirty categorical data
- Invalid numeric values
- Incorrect date formats

### Skills Used
- `drop_duplicates()`
- `fillna()`
- `replace()`
- `pd.to_numeric()`
- `pd.to_datetime()`
- String cleaning using `.str`
